BLENDER BRANCH - creates simple snake object and keyframes animation based on MATLAB outputs.

The winding axis is the Z-axis. Uses R<=a calculation for R_vec.

Run "exportBlenderData.m" to create a csv file containing the data that will be sent to Blender.
Update path to "blenderData.csv" in "makeSpiral.py", then run "makeSpiral.py" in Blender.
